# BrightTV Viewership Analysis and CVM Growth Strategy

## Project Overview
This project focused on cleaning and consolidating complex subscriber and engagement data into a high performance analytics environment using Databricks and SQL. The primary objective was to resolve data inconsistencies between subscriber profiles and viewership logs to create a unified source of truth for the Customer Value Management team.

The core mission is to assist BrightTV in growing its subscription base for the current financial year by turning technical data into a strategic roadmap.

## Key Findings and Insights
* **Provincial Engagement Trends:** Analysis revealed that viewership peaks in specific regions like Gauteng and Western Cape during evening prime time, while rural provinces show higher engagement with mobile-first content during morning hours.
* **Churn Indicators:** Data showed a direct correlation between low consumption days and subscriber churn. By identifying these quiet periods, the team can now deploy targeted push notifications or content highlights to re-engage users before they cancel.
* **Demographic Consumption:** Specific South African demographics show a strong preference for local content over international acquisitions. Tailoring recommendations based on these cultural preferences is expected to increase platform loyalty.
* **Peak Usage Windows:** The transition from afternoon to evening segments represents the highest drop-off risk. Optimizing the content transition during this window can help maintain a steady flow of viewers into the high-revenue evening slots.

## Key Objectives
* **Data Integration:** Unified disparate datasets to identify exactly what drives engagement across different South African demographics.
* **Churn Mitigation:** Investigated periods of low consumption to provide evidence based content recommendations to boost activity.
* **Growth Identification:** Analyzed viewing patterns across morning, afternoon, and evening segments to pinpoint opportunities for increasing platform usage.
* **Strategic Programming:** Provided insights to help the CVM team suggest the right content to retain current users and attract new subscribers.

## Tools and Technologies
* **SQL (Databricks):** Used for data cleaning, handling null values, and performing complex joins to merge subscriber profiles with viewership logs.
* **Excel:** Developed Pivot Tables to analyze provincial viewership trends and subscriber demographics.
* **Miro:** Created project mind maps and data flow diagrams.
* **Canva:** Designed a project timeline and the final executive dashboard.
* **Lovable:** Built interactive data visualizations and prototypes for the viewership dashboard.

## Project Structure
* 📁 **data/**: Contains the raw viewership CSV and the cleaned Excel analysis with Pivot Tables.
* 📁 **scripts/**: Contains the SQL queries used for data engineering and viewership calculations.
* 📁 **planning/**: Includes the Miro mind map and Canva project timeline.
* 📁 **output/**: Contains the final visual dashboard and the strategic presentation.
