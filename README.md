# BrightTV Viewership Analysis & CVM Growth Strategy

## Project Overview
This project focused on cleaning and consolidating complex subscriber and engagement data into a high-performance analytics environment using **Databricks and SQL**. The primary objective was to resolve data inconsistencies between subscriber profiles and viewership logs to create a unified "source of truth" for the Customer Value Management (CVM) team.

The core mission is to assist BrightTV in growing its subscription base for the current financial year by turning technical data into a strategic roadmap.

## Key Objectives
* **Data Integration:** Unified disparate datasets to identify exactly what drives engagement across different South African demographics.
* **Churn Mitigation:** Investigated periods of low consumption to provide evidence-based content recommendations to boost activity.
* **Growth Identification:** Analyzed viewing patterns (Morning, Afternoon, and Evening) to pinpoint opportunities for increasing platform usage.
* **Strategic Programming:** Provided insights to help the CVM team suggest the right content to retain current users and attract new subscribers.

## Tools & Technologies
* **SQL (Databricks):** Used for data cleaning, handling null values, and performing complex joins (Full Outer Joins) to merge subscriber profiles with viewership logs.
* **Excel:** Developed Pivot Tables to analyze provincial viewership trends and subscriber demographics.
* **Miro:** Created project mind maps and data flow diagrams.
* **Canva:** Designed a project timeline (Gantt Chart) and the final executive dashboard.

## Project Structure
* 📁 **data/**: Contains the raw viewership CSV and the cleaned Excel analysis with Pivot Tables.
* 📁 **scripts/**: Contains the SQL queries used for data engineering and viewership calculations.
* 📁 **planning/**: Includes the Miro mind map and Canva project timeline.
* 📁 **output/**: Contains the final visual dashboard and the 6-slide strategic presentation.
